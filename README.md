## Data Format
Loading in a single putative neuron dataset will give you a 'data' variable. This variable is formatted like this:

| Index | Column 1  | Column 2 |
|-------|-----------|----------|
| 1     |           | bin      |
| 2     | rm_con    | rm bin   |
| 3     | mtf_con   | mtf bin  |
| 4     | strf_con  | strf bin |
| 5     | rvf       | schr     |
| 6     |           | Oboe        |
| 7     |           | Bassoon     |
| 8     |           | Other nt    |
